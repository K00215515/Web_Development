Design elements used

The two background colours are separated by using a div in the html file. The div would be called by a name such as "page" which can be used to create a background colour that is different from the body background.

The navigation bar and the style of the header was taken from a previous assignment as inspiration. The navigation bar has a display of inline which is just vertical. The style of the header is only using the font-style element with underline and overline. The website is adaptable if the screen is small such as 300px like a smartphone.
